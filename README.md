# Royal Barber AI Project

## Project Concept

Royal Barber is a luxury barbershop website created using AI-powered tools and prompt engineering techniques.

## AI Tools Used

* NanoBanana (website design)
* devv.ai (website generation)
* VEO3 (promo video)
* Suno (promo music)

## Features

* Premium luxury design
* Responsive layout
* VIP packages section
* Barber team showcase
* Booking form
* Gallery section
* Modern UI/UX

## Prompt Engineering Techniques

* Role prompting
* Style conditioning
* Constraint-based prompting
* Iterative refinement

## Project Status

Website successfully generated and deployed using AI workflow.
